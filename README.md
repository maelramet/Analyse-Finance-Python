# Analyse-Finance-Python
Projet Finance &amp; Python : Analyse Portefeuille vs CAC 40 Développement d’un outil quantitatif (Pandas, yFinance) évaluant la performance d’actifs via le modèle MEDAF. • Calcul d'indicateurs clés : Alpha de Jensen, Beta et Volatilité. • Analyse critique : Impact des taux de change (EUR/USD) et corrélation des marchés sur 5 ans. • Stack : Python
